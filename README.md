## ğŸ§  About

Serac is a Pomodoro focus tool designed to make life easier for people with ADHD and other neurodivergent individuals.  
It helps users overcome common digital distractions by combining customizable focus timers with automatic blocking of distracting websites, enabling more comfortable, independent, and confident interaction with technology.

Built for **Grizzly Hacks II (February 2026)** with the mission to create software that reduces digital barriers and improves daily productivity for people with disabilities.
